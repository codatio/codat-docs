## Types de données requis

Tous les types de données pertinents pour la génération de métriques de commerce sont activés automatiquement lorsque la solution Lending est activée pour votre client Codat. En cas de problèmes lors de la génération des métriques, vérifiez que les types de données suivants n'ont pas été accidentellement désactivés :

- Commerce - Company Info
- Commerce - Customers
- Commerce - Orders
