Avec Enhanced Financials, Codat suggère automatiquement une catégorie pour chaque compte qui apparaît dans les états Enhanced Profit and Loss et Balance Sheet.

Chaque compte se voit attribuer l'une des 270+ catégories de Codat. Si vous n'êtes pas d'accord avec la catégorie suggérée, vous pouvez la modifier pour une catégorie plus appropriée. Cela n'affecte que le champ de catégorie dans Lending et ne provoque aucun changement dans le logiciel de comptabilité sous-jacent.

Vous pouvez visualiser toutes les catégories disponibles proposées pour les comptes et, le cas échéant, les recatégoriser dans le <a href="https://app.codat.io/" target="_blank">Portal Codat</a>.

1. Naviguez vers **Companies**, puis cliquez sur l'entreprise qui nécessite une révision de catégorisation. Sélectionnez **Products > Lending** dans le menu latéral. Enfin, cliquez sur **Categorize accounts**.

![Une image de la vue du portail Lending et du bouton de catégorisation](/img/lending/acct-categorization-v3-1.png)

2. Vous pouvez voir les catégories pour chaque compte. Celles-ci sont ordonnées par _impact_ par défaut, qui est déterminé par le solde actuel du compte et notre confiance dans notre catégorisation automatique. Les comptes avec un _impact_ élevé affectent davantage les [Enhanced Financials](/lending/enhanced-financials/overview), il vaut donc la peine de confirmer ces comptes.

![Une image de la vue de catégorisation Lending dans le Portal](/img/lending/acct-categorization-v3-2.png)

3. Pour modifier la catégorie d'un compte, sélectionnez les comptes en utilisant la case à cocher et cliquez sur **Recategorize**.

   Choisissez une catégorie appropriée parmi les cinq niveaux proposés et cliquez sur **Recategorize**. Cela enregistre la catégorie nouvellement attribuée. Une fois que vous avez confirmé ou modifié un compte, l'_impact_ passera à zéro.

![Une image de la vue de catégorisation Lending dans le Portal avec un compte en cours de recatégorisation](/img/lending/acct-categorization-v3-3.png)

C'est tout! Les états Enhanced Profit and Loss et Enhanced Balance Sheet retourneront la catégorie mise à jour pour les comptes à l'avenir.
