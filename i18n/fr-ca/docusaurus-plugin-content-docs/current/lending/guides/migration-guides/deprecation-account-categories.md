## Que se passe-t-il?

Le **10 juillet 2023**, nous déprécierons la version ancienne des catégories de comptes et toutes les fonctionnalités associées, y compris :

- Account categories
- Enhanced Balance Sheet (version ancienne uniquement)
- Enhanced Profit and Loss (version ancienne uniquement)
- Financial metrics
- Marketing metrics

Pour comprendre ce qui changera exactement et comment tirer parti des alternatives (y compris les instructions de migration), consultez les pages ci-dessous :

- [Clients ayant construit avec le produit Lending](/lending/overview)
- [Clients utilisant Assess dans le Portal](/lending/overview)
- [Clients utilisant l'extraction de données vers Excel dans le Portal](/lending/overview)
