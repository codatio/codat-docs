## Que se passera-t-il lors de la dépréciation?

La façon dont vous téléchargez les données via le bouton **Export data** changera. Celui-ci ne contiendra plus les feuilles suivantes :

- Account categories
- Categorized account balances

![Capture d'écran du bouton d'exportation de données](/img/lending/data-export.png)

Note : Tous les autres onglets dans l'exportation de données et le rapport d'audit continueront d'être pris en charge.

## Instructions pour la migration

| Fonctionnalité                                                                                                                                     | Alternative                                                                                                                                                                                                                                                                                                                                                                                                  |
| :------------------------------------------------------------------------------------------------------------------------------------------ | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Si vous utilisez les feuilles suivantes dans l'exportation Excel : <br/>1. Feuille Account categories <br/>2. Feuille Categorized account balances | Les dernières catégories de comptes peuvent être téléchargées en Excel dans Assess. Pour ce faire : <br/> 1. Connectez-vous au Portal Codat<br/> 2. Sélectionnez une entreprise <br/> 3. Sous les produits, cliquez sur **Assess**, puis sur **Reports** <br/> 4. Une nouvelle option **Enhanced Financials** est disponible pour être générée et téléchargée. <br/><br/> Si vous n'avez pas accès à Assess, veuillez contacter votre gestionnaire de compte pour l'activer. |
