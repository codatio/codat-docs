## Options alternatives

### Visualiser les états financiers et calculer les ratios

1. Connectez-vous au **Portal Codat**.

2. Sélectionnez une entreprise sur la page **Companies**.

3. Sous Products, cliquez sur **Assess**, puis sur **Reports**.

4. Une nouvelle option **Assess Report** est disponible pour être générée et téléchargée.
5. Ouvrez le fichier téléchargé dans Microsoft Excel.

6. Utilisez les tableaux croisés dynamiques et les graphiques pour visualiser et interroger les états financiers.
7. Additionnez les soldes par catégorie pour calculer les ratios financiers.

![Capture d'écran des états financiers dans Excel](/img/lending/financial-statements-excel.png)

### Catégoriser les comptes

Il existe une nouvelle version de l'écran de catégorisation des comptes disponible pour les dernières catégories de comptes. Pour y accéder :

1. Sélectionnez **Assess** dans le menu de gauche
2. Cliquez sur **Categorize accounts**

Il existe une nouvelle version de cet écran disponible pour les dernières catégories de comptes. Changez simplement le bouton **Version** en haut à droite de l'écran à <i>Version 3</i>

![Capture d'écran de l'écran de catégorisation des comptes dans le portail](/img/lending/account-categorization-v3.png)
