Vous pouvez obtenir un accès en temps réel aux données de votre client et aux perspectives les plus à jour si celui-ci autorise une connexion numérique entre Codat et son logiciel comptable.

## Parcours de connexion

Lors de l'[intégration du client](/spend-insights/guides/onboard-customer), vous avez partagé un **Link URL** avec eux qui initie un parcours de connexion numérique. Ce parcours comprend les étapes suivantes :

1. Le client clique sur le Link URL et est dirigé vers l'écran de sélection du logiciel.
2. Le client recherche et sélectionne le logiciel qu'il souhaite connecter.
3. Le client examine les détails des données qu'il partage et initie la connexion.

   Cela ouvre le logiciel du client dans un nouvel onglet. Il doit alterner entre l'onglet du logiciel et l'onglet du parcours de connexion pour compléter le processus.

4. Le client se connecte au logiciel et approuve la demande de partage de données.

   Les identifiants du client restent sécurisés et ne sont pas partagés avec Codat ou votre organisation.

5. Lorsque la connexion est confirmée, le client est invité à terminer et quitter le flux.

## Guides des logiciels

**Spend Insights** prend en charge la diffusion continue de données avec les logiciels listés ci-dessous. Pour assurer une connexion réussie, nous fournissons des instructions d'accès pour toutes ces plateformes. Certaines sont plus complexes que d'autres, donc leurs parcours de connexion prennent plus de temps.

Cliquez sur la tuile du logiciel pour accéder aux instructions de connexion étape par étape ou visitez notre [Centre d'aide](https://help.codat.io). Vous pouvez partager les guides avec votre client ou utiliser les instructions pour fournir un soutien en personne ou par téléphone.

:::tip Liste de vérification du client

Lorsque vous invitez le client à établir une connexion numérique, faites-lui savoir que :

- Il devra suivre le Link URL que vous avez partagé avec lui.
- Il devra avoir un accès Administrateur ou équivalent dans son logiciel comptable.
- Il devra partager son écran avec vous s'il demande un soutien téléphonique.

Vous pouvez également partager de manière proactive les [réponses aux questions courantes](/spend-insights/resources/customer-faqs) avec eux.

:::
