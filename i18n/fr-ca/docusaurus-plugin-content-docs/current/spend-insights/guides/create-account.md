Vous gérerez vos clients et leurs perspectives de dépenses en utilisant le tableau de bord disponible sur le [Portail Codat](https://app.codat.io). Suivez les directives ci-dessous pour créer et gérer votre compte et votre mot de passe avec Codat.

## Créer un compte

Pour commencer, vous recevrez un courriel de `no-reply@mail.codat.io` avec une invitation à rejoindre l'instance Codat de votre organisation. Cliquez sur le lien d'inscription et complétez les étapes suivantes :

1. Entrez l'adresse courriel dans le champ **Email**. Ce sera votre courriel de connexion Codat.
2. Créez un mot de passe et entrez-le dans le champ **Password**.
3. Entrez votre nom complet dans le champ **Full name** et complétez votre inscription.

Vous serez immédiatement redirigé vers le tableau de bord Spend Insights du [Portail Codat](https://app.codat.io). Nous vous recommandons de l'enregistrer comme signet pour y accéder facilement plus tard.

Selon votre rôle, vous pourrez effectuer les actions suivantes dans le Portail :

- Créer un client
- Envoyer une demande de partage de données au client
- Télécharger le fichier d'analyse des dépenses

:::tip Vous n'avez pas reçu le courriel d'inscription?

Si vous ne trouvez pas le courriel, vérifiez votre dossier Spam ou Courrier indésirable pour les courriels provenant de `no-reply@mail.codat.io`. Si vous ne parvenez toujours pas à le trouver, communiquez avec nous.

:::
