Notre [Portail Codat](https://app.codat.io/) est votre guichet unique pour consulter les clients et les relations dont vous êtes responsable. Ici, les clients de votre organisation sont représentés en tant qu'entreprises.

Dans ce guide, nous couvrirons les façons dont vous pouvez créer de nouveaux clients, gérer ceux existants et demander des données financières de leur part.

## Créer un client

Vous pouvez créer votre client dans Codat à partir de l'onglet _Spend Insights_ ou de l'onglet _Companies_. Dans notre exemple, nous commençons sur l'onglet _Spend Insights_.

1. Cliquez sur le bouton **Create company**. Cela ouvrira une fenêtre _Create a company_.

   ![Une image de l'interface utilisateur du Portail Codat avec trois entreprises listées comme exemples et le bouton Create company mis en évidence](/img/spend-insights/si-create-company-button.png)

2. Dans la fenêtre _Create a company_, entrez le nom de l'entreprise de votre client dans le champ **Company name**. Vous pouvez ajouter une description optionnelle dans le champ **Description** et ignorer les **Company tags**.

   :::caution Ajout de produits

   Vous pourriez voir le menu déroulant **Add products** lors de la création d'une entreprise à partir de l'onglet _Companies_. Sélectionnez **Spend Insights** et continuez la création de l'entreprise.

   :::

   ![Une image de la fenêtre Create a company avec le champ Company name rempli avec la valeur Hairosmith et la description de l'entreprise indiquée comme Rockstar-quality fringes and cuts. Le bouton Create company est mis en évidence](/img/spend-insights/si-create-company-window.png)

3. Cliquez sur **Create company** pour terminer la création de votre client dans Codat.

   ![Une image de la fenêtre Create a company avec un message de succès indiquant que Hairosmith a été créé. L'écran affiche des options pour copier un Link URL, copier un modèle de courriel et créer une autre entreprise.](/img/spend-insights/si-create-company-created.png)

:::tip Prochaines étapes

Vous avez créé avec succès votre client en tant qu'entreprise dans Codat. Ensuite, vous pouvez :

- Cliquer sur **Create another company** pour ajouter d'autres clients à Codat.
- Utiliser le **Link URL** ou le **Email template** pour intégrer votre client. Nous expliquons ces options plus en détail dans notre guide [Intégrer un client](/spend-insights/guides/onboard-customer#onboard-customer).
- [Gérer les détails](/spend-insights/guides/onboard-customer#manage-customers) des clients que vous avez créés.
- Fermer la fenêtre _Create a company_ et naviguer vers l'onglet _Spend Insights_.

:::

## Intégrer un client

Vous pouvez initier des demandes pour les informations de comptes fournisseurs de vos clients de plusieurs façons :

- En utilisant le bouton **Request data** dans l'onglet _Spend Insights_.
- En utilisant le bouton **Request data** dans l'onglet _Companies_.

Les étapes pour demander ces informations sont les mêmes pour toutes ces options :

1. Cliquez sur le bouton **Request data** à côté du nom du client concerné pour ouvrir la fenêtre _Onboarding_.

   ![Une image de l'onglet Companies avec quatre entreprises listées comme exemples. Un bouton Request data est mis en évidence à côté d'une entrée d'entreprise appelée Hairosmith.](/img/spend-insights/si-manage-company-request-info.png)

2. Dans la fenêtre _Onboarding_, choisissez comment vous souhaitez demander les données à votre client :
   - Copiez le **Link URL** si vous souhaitez envoyer un lien de connexion directe à votre client ou utiliser le lien dans votre propre communication.
   - Copiez le **Email template** si vous souhaitez obtenir de l'aide pour fournir un contexte supplémentaire à votre client. Vous pouvez consulter le modèle en cliquant sur l'icône du menu déroulant.

   ![Une image de la fenêtre Onboarding pour une entreprise appelée Hairosmith est ouverte. Deux options, Link URL et Email template, sont visibles. Chaque option a un bouton de copie à côté qui est mis en évidence. L'option Email template a également une icône de menu déroulant mise en évidence à côté. En arrière-plan, le contenu de l'onglet Companies est visible.](/img/spend-insights/si-request-information-options.png)

   :::info Demander des données lors de la création du client

   Vous verrez le **Link URL** et le **Email template** lorsque vous [créez votre client](/spend-insights/guides/onboard-customer#create-customer) dans Codat pour la première fois.

   Selon les processus d'affaires de votre organisation, vous pouvez demander des données immédiatement après la création du client ou le faire plus tard en suivant ce processus.

   :::

3. Selon les méthodes de travail de votre organisation, partagez le Link URL ou le contenu du courriel avec votre client. Vous pouvez également partager de manière proactive les [réponses aux questions courantes](/spend-insights/resources/customer-faqs) avec eux.

:::tip Prochaines étapes

Vous avez demandé les informations sur les fournisseurs et les dépenses de votre client. Ensuite, votre client doit compléter un parcours de connexion pour les fournir à votre organisation. Selon le logiciel qu'il utilise, il peut autoriser une connexion numérique ou téléverser des fichiers contenant les données de fournisseurs et de factures.

Vous pouvez accompagner votre client dans ce parcours en utilisant les guides suivants :

- [**Diffusion de données en continu**](/spend-insights/resources/link-software) : Aidez votre client à établir une connexion continue avec son logiciel comptable.
- [**Téléversement intelligent de fichiers**](/spend-insights/resources/link-file) : Guidez votre client dans le partage de fichiers contenant ses informations financières.

:::

## Gérer les clients

Vous pouvez consulter tous les clients existants dans l'onglet _Spend Insights_ ou l'onglet _Companies_.

Pour modifier ou supprimer vos clients existants, accédez à l'onglet _Companies_. Utilisez l'icône **Crayon** à côté du nom de l'entreprise pour modifier ses détails ou l'icône **Corbeille** pour la supprimer.

Si vous supprimez une entreprise, cela supprimera également les données que vous avez précédemment reçues de celle-ci. Par conséquent, vous ne pourrez plus accéder aux rapports de cette entreprise dans le Portail. Cette opération est irréversible.

![Une image de l'onglet Companies avec quatre entreprises listées comme exemples. Une icône de crayon et une icône de corbeille sont visibles et mises en évidence à côté d'une entrée d'entreprise appelée Hairosmith.](/img/spend-insights/si-manage-company.png)

:::tip Consulter les perspectives de dépenses

Une fois que votre client partage ses informations financières avec vous, vous pouvez [consulter l'analyse des dépenses et les perspectives](/spend-insights/guides/analyze-spend) que Codat a préparées à partir de ces données.

:::
