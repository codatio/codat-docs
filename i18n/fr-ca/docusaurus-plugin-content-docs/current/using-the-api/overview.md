Bien que le portail offre un moyen simple d'explorer les données, vous devrez utiliser l'API Codat pour tirer le meilleur parti des données de votre entreprise.

## API

Nous avons regroupé les fonctionnalités de Codat principalement par _solution_, avec notre _API Plateforme_ fournissant les fonctionnalités de base utilisées dans toutes nos solutions. Vous pouvez explorer chaque API et ses endpoints dans nos références API.

| API de solution                                 | Description                                                                                                                          |
| :---------------------------------------------- | :----------------------------------------------------------------------------------------------------------------------------------- |
| **[Bank Feeds](/bank-feeds-api)**               | Configurez des flux bancaires depuis les comptes de votre application vers les logiciels comptables pris en charge.                  |
| **[Lending](/lending-api)**                      | Prenez des décisions de crédit appuyées par des données financières améliorées, des métriques, des rapports et des fonctionnalités d'intégrité des données. |
| **[Expenses](/sync-for-expenses-api)**           | Simplifiez la réconciliation des dépenses depuis la carte d'entreprise et les systèmes de gestion des dépenses vers le logiciel comptable de vos clients. |
| **[Bill Pay](/sync-for-payables-v2-api#/)**      | Facilitez la gestion et le paiement des fournisseurs par vos clients depuis une interface unique.                                    |

Explorez les [spécifications OpenAPI](https://github.com/codatio/oas) sous-jacentes.
