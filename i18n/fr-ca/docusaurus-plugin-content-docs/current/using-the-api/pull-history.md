Bientôt disponible!

## Dans le portail

Lors de la consultation d'une entreprise dans le portail, vous pouvez consulter l'**Historique de lecture** pour vérifier le statut des synchronisations de données de lecture et d'écriture précédentes par type de données.

`Entreprises > Entreprise > Historique des données > Historique de lecture`

Lorsqu'une lecture de données est initialisée, des éléments apparaîtront ici pour chaque type de données en cours de lecture.

Vous pouvez voir :

- Type - nom du type de données, avec le nom de l'intégration en dessous
- ID - l'identifiant de l'opération de lecture pour ce type de données
- Date de demande
- Temps d'exécution
- Statut

Lorsqu'une opération de lecture rencontre une erreur de validation, vous pouvez cliquer sur le statut pour voir plus d'informations.
