Vous pouvez tirer parti des informations exploitables sur les fournisseurs et les activités de dépenses de notre solution [Spend Insights](/spend-insights/overview) basée sur les données de comptes fournisseurs de notre intégration <a className="external" href="https://www.workday.com/" target="_blank">Workday</a>.

Workday est un fournisseur de logiciels de gestion financière, de gestion du capital humain et de systèmes d'information pour étudiants destiné au segment du marché intermédiaire.

## Démarrage

Aucune configuration n'est requise pour commencer à utiliser Workday. Activez Workday dans le [Portail Codat](https://app.codat.io) pour connecter les données de vos premiers clients Workday.

Si vous avez besoin d'une instance de test Workday, vous devrez passer par le programme de partenariat de Workday, qui comprend une formation approfondie dans le cadre du processus d'approbation. Contactez votre gestionnaire de compte Codat pour en discuter.

### Activer l'intégration Workday

1. Dans le Portail Codat, accédez à la page <a className="external" href="https://app.codat.io/settings/integrations/accounting" target="blank">**Intégrations comptables**</a>.
2. Localisez **Workday** et cliquez sur le bouton à bascule pour activer l'intégration.

Vous pouvez également cliquer sur **Gérer** pour afficher la page des paramètres de l'intégration, puis activer l'intégration à partir de là.
