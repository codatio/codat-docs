Vous pouvez choisir si les connecteurs sur site Sage 50 (RU et Irlande) et QuickBooks Desktop s'exécutent une seule fois ou restent installés sur les machines de vos clients.

Deux options de configuration de connecteur sont disponibles.

- **Installer sur la machine du client (par défaut) :** Après l'installation du connecteur, les données du client sont périodiquement mises à jour dans le portail Codat. Le connecteur reste installé sur l'ordinateur de l'utilisateur.
- **Ne pas installer / synchronisation unique :** Après le téléchargement des données depuis le logiciel comptable du client, le connecteur se supprime lui-même de l'ordinateur de l'utilisateur. Une synchronisation unique est effectuée lorsque l'utilisateur exécute le connecteur.

Si vous optez pour l'installation du connecteur, vous pourrez actualiser les données de vos clients à l'avenir, sans aucune action de leur part.

La configuration du connecteur sur site est définie dans le Portail Codat au niveau de l'intégration. Vous pouvez remplacer les paramètres pour des entreprises individuelles si nécessaire.

## Configurer les options de configuration du connecteur au niveau de l'intégration

1. Ouvrez le [Portail Codat](https://app.codat.io) et connectez-vous.
2. Dans la barre de navigation, sélectionnez **Paramètres > Intégrations > Comptabilité**.
3. Trouvez soit les intégrations **QuickBooks Desktop** soit **Sage 50 (RU et Irlande)** puis sélectionnez **Configurer**.
4. Choisissez l'une des options suivantes :
   - **Ne pas installer / synchronisation unique**
   - **Installer sur la machine du client**

5. Cliquez sur **Enregistrer**.
