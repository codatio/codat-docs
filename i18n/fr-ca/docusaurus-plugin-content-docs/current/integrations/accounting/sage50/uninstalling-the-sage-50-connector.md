Si un utilisateur souhaite arrêter de synchroniser ses données avec Sage 50, il peut désinstaller le connecteur Sage 50 Accounts de son ordinateur.

## Désinstaller le connecteur Sage 50 Accounts depuis l'icône de la barre système

Pour désinstaller le connecteur Sage 50 Accounts, l'utilisateur peut cliquer avec le bouton droit sur l'icône de la barre système du connecteur, puis sélectionner **Désinstaller le connecteur Sage 50 Accounts**.

![Text](/img/old/a829482-RightClickMenu.png "Désinstaller le connecteur Sage 50 Accounts sélectionné dans le menu contextuel de la barre système.")

:::note

Si vous avez configuré l'icône de votre entreprise dans le portail Codat, vous verrez cette icône dans la barre système et non l'icône Codat.
:::

Ensuite, l'utilisateur est invité à confirmer qu'il souhaite désinstaller le connecteur Sage 50 Accounts. S'il clique sur **Désinstaller**, le connecteur sera désinstallé et l'entreprise Codat avec laquelle il synchronisait deviendra _Non autorisée_.

![Text](/img/old/58b407a-uninstall_button.png "Boîte de dialogue Désinstaller le connecteur Sage 50 Accounts.")

## Désinstallation si le connecteur est fermé avant la première liaison

Le connecteur Sage 50 Accounts sera désinstallé si l'utilisateur ferme le connecteur avant d'avoir complété toutes les étapes du flux de liaison (voir [Configurer le connecteur Sage 50 Accounts](/integrations/accounting/sage50/installing-the-sage-50-connector).

Lorsque l'icône de fermeture est cliquée, le connecteur est désinstallé et l'entreprise Codat qui allait être liée deviendra _PendingAuth_.

![Text](/img/old/624728d-uninstall_connector_-_X_button_on_link_flow.jpg "Assistant Ajouter une entreprise avec l'icône de fermeture mise en évidence.")
