Vous pouvez lire les données comptables de <a className="external" href="https://www.clearbooks.co.uk/" target="_blank">Clear Books</a> en utilisant notre intégration Clear Books.

Clear Books fournit un logiciel de comptabilité en ligne pour les petites entreprises basées au Royaume-Uni.

## Configurer l'intégration

Aucune inscription d'application n'est requise pour que l'intégration Clear Books fonctionne. Lors de la liaison de leurs données, l'utilisateur PME est invité à saisir sa clé API comme indiqué :

<img src="/fr-ca/img/old/0c9d3d9-Clear_Books_Link.png" />

Pour trouver sa clé API, l'utilisateur PME effectue les étapes suivantes :

1. Se connecte à son compte Clear Books.

2. Sélectionne **Settings > Configure system > API keys** dans la barre de menu.

   ![Clear Book: L'utilisateur PME sélectionne API keys dans Settings > Configure system dans la barre de menu.](/img/old/00c84fa-clear-books_select-api-key.png)

3. Sélectionne **Provide required information**, puis fournit les informations suivantes :
   1. Sous **Intended API usage**, sélectionne la deuxième option : **I want to connect Clear Books with a piece of third-party software, exclusively for my own use**.
   2. Sous **Third-party software information**, saisit le nom de votre application ou site web dans la première case et "Codat" dans la deuxième case.

Sa clé API est maintenant disponible pour être copiée depuis la page **API keys** (sous **SOAP API**).

<img
  src="/fr-ca/img/old/c176f58-clear-books_api-keys-page-highlighted.png"
  alt="Clear Books : Page API keys avec la clé API de l'utilisateur PME mise en évidence sous 'SOAP API'."
/>

### Activer l'intégration Clear Books

1. Dans le portail Codat, accédez à la page <a className="external" href="https://app.codat.io/settings/integrations/accounting" target="blank">**Intégrations comptables**</a>.
2. Localisez **Clear Books** et cliquez sur le bouton à bascule pour activer l'intégration.

Vous pouvez également cliquer sur **Manage** pour afficher la page des paramètres de l'intégration, puis activer l'intégration à partir de là.
