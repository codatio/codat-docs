Vous pouvez synchroniser les données comptables avec <a  class="external" href="https://www.sage.com/en-gb/sage-business-cloud/intacct/" target="_blank">Sage Intacct</a> en utilisant notre intégration Sage Intacct.

Sage Intacct est une plateforme infonuagique destinée aux entreprises de marché intermédiaire en croissance.

Pour plus de détails sur les types de données et les opérations pris en charge, consultez [Référence de l'intégration Sage Intacct](/integrations/accounting/sage-intacct/sage-intacct-integration-reference).

## Configurer l'intégration

Consultez [Configurer l'intégration Sage Intacct](/integrations/accounting/sage-intacct/accounting-sage-intacct-setup) pour apprendre comment configurer et activer l'intégration.
