Exploitez les informations exploitables de nos [Spend Insights](/spend-insights/overview) basées sur les données comptables de notre intégration <a className="external" href="https://www.microsoft.com/en-gb/dynamics-365/products/finance" target="_blank">Dynamics 365 Finance & Operations</a> (F&O).

Dynamics 365 F&O est une solution de planification des ressources d'entreprise (ERP) basée sur le cloud qui couvre les principaux processus opérationnels de chaîne d'approvisionnement et de finance.

## Pour commencer

Aucune configuration n'est requise pour commencer à utiliser Dynamics 365 F&O. Contactez votre gestionnaire de compte ou votre gestionnaire de réussite client pour activer l'intégration et inviter vos clients à connecter leurs données.

### Activer l'intégration Dynamics 365 F&O

Si vous devez gérer l'intégration, accédez à **Settings > Integrations > Accounting** pour afficher la page [Intégrations comptables](https://app.codat.io/settings/integrations/accounting) dans le portail Codat.

Là, localisez **Dynamics 365 Finance & Operations** et utilisez le bouton à bascule pour gérer l'intégration. Alternativement, cliquez sur **Manage** pour afficher la page des paramètres de l'intégration et utiliser le bouton à bascule à partir de là.
