Vous pouvez synchroniser les données comptables et bancaires avec <a className="external" href="https://www.myob.com/" target="_blank">MYOB Business</a> en utilisant notre intégration MYOB Business.

:::info MYOB Business, AccountRight et Essentials

Notre intégration MYOB s'applique aux utilisateurs utilisant les produits suivants :

- MYOB Business
- MYOB AccountRight
- MYOB Essentials
  :::

Pour plus de détails sur les types de données pris en charge, consultez [Référence de l'intégration MYOB](/integrations/accounting/myob/myob-integration-reference).

## Configurer l'intégration

Consultez [Configurer l'intégration MYOB](/integrations/accounting/myob/accounting-myob-setup) pour apprendre comment configurer et activer l'intégration.

:::info MYOB AccountRight ou Essentials?

MYOB a migré son API Essentials vers une nouvelle version. La nouvelle version utilisera l'API AccountRight. Vous pouvez en savoir plus sur la mise à jour dans le [centre développeur MYOB](https://developer.myob.com/api/myob-business-api/api-overview/getting-started/).

Le nouvel Essentials utilisera la même API que leur plateforme AccountRight Live. Par conséquent, vous n'avez besoin d'effectuer la configuration décrite ci-dessous qu'une seule fois pour accéder aux deux logiciels comptables.

Certains utilisateurs peuvent encore être sur l'ancienne version non migrée de MYOB Essentials qui utilise une API héritée. Si votre client ne peut pas se connecter en utilisant la nouvelle intégration unifiée, veuillez utiliser notre [formulaire de demande de support](https://codat.zendesk.com/hc/en-gb/requests/new) pour créer un ticket.
:::

## Versions prises en charge

:::info Utilisation de l'intégration MYOB Business
Les données de l'entreprise doivent être stockées dans le cloud, et non sur une machine locale.
:::

- Tous les niveaux sont pris en charge : **Standard**, **Plus** et **Premier**.
- Nous sommes en mesure de lier les entreprises AccountRight stockées dans le cloud tant que le fichier de l'entreprise fonctionne sur une version prise en charge par MYOB.
