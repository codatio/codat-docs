Vous pouvez synchroniser les données comptables avec <a href="https://www.zoho.com/books/" target="_blank">Zoho Books</a> en utilisant notre intégration Zoho Books.

Pour plus d'informations sur les types de données pris en charge, consultez [Référence de l'intégration Zoho Books](zoho-books-integration-reference).

## Configurer l'intégration

Consultez [Configurer l'intégration Zoho Books](/integrations/accounting/zoho-books/accounting-zohobooks-setup) pour apprendre comment configurer et activer l'intégration.

:::caution Ensembles de données réduits
Par défaut, des ensembles de données réduits sont lus pour les types de données Bills, Credit Notes, Customers et Invoices. Pour plus d'informations, consultez [Limites de Zoho Books et lectures de données réduites](zoho-book-limits).
