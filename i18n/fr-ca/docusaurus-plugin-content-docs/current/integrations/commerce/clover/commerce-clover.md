<p>
  <a className="external" href="https://uk.clover.com/" target="_blank">
    Clover
  </a>
  facilite la gestion de leur entreprise par les commerçants, en combinant les fonctionnalités de point de vente
  et de traitement des paiements dans une seule solution logicielle.
</p>

Utilisez l'API Commerce de Codat avec Clover pour vous connecter de manière sécurisée, récupérer et visualiser les transactions commerce de vos clients.

## Configurer l'intégration

Voir [Configurer l'intégration Clover](/integrations/commerce/clover/set-up-your-clover-integration) pour apprendre comment configurer et activer l'intégration.

:::caution Exigences du fournisseur sous-jacent
Clover a des restrictions sur les cas d'utilisation pour lesquels ils approuveront des applications. Avant de configurer l'intégration, consultez les [Clover App Market Developer Program Policies](https://www.clover.com/app-market-policies) pour vous assurer que votre cas d'utilisation est pris en charge.
