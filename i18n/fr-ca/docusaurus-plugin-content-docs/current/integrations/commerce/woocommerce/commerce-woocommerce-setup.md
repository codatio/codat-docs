Avant de pouvoir lire les données commerce des entreprises utilisant WooCommerce, vous devez configurer l'intégration.

Vous devrez :

- Vous connecter au [portail Codat](https://app.codat.io/).
- Activer l'intégration WooCommerce.
- Vérifier vos paramètres de synchronisation.

L'inscription auprès de WooCommerce n'est pas requise pour configurer et activer l'intégration.

## Activer l'intégration WooCommerce

1. Dans le portail Codat, accédez à **Paramètres > Intégrations > Commerce** pour afficher la page [Intégrations Commerce](https://app.codat.io/settings/integrations/commerce).
2. Localisez **WooCommerce**, puis sélectionnez **Configurer** pour afficher la page **Paramètres d'intégration**.
3. Entrez le **Nom de l'application** à afficher dans WooCommerce. Celui-ci est affiché lorsqu'un marchand lie sa boutique et qu'on lui demande la permission de la connecter. Par exemple, en utilisant le nom d'application « Codat » :

   ![WooCommerce store connection flow](/img/integrations/commerce/woocommerce/woocommerce-app-connection-flow.png "La boîte de dialogue de connexion de boutique WooCommerce utilisant un nom d'application de Codat")

4. Choisissez quel [type d'accès aux données d'entreprise](/core-concepts/data-type-settings) vous souhaitez pour cette intégration : ponctuel ou continu.
5. Cliquez sur **Enregistrer**.
6. Dans la boîte de dialogue qui apparaît, cliquez sur **Activer WooCommerce**.

Vous pouvez également cliquer sur **Gérer** pour afficher la page des paramètres de l'intégration, puis activer l'intégration à partir de là.

## Vérifier vos paramètres de synchronisation dans le portail Codat

S'il s'agit de votre première intégration commerce, mettez à jour vos [paramètres de type de données](/integrations/commerce/commerce-sync-settings) pour activer les types de données commerce.
