[WooCommerce](https://woocommerce.com/) est un logiciel eCommerce personnalisable et open source construit sur WordPress.

Utilisez l'API Commerce de Codat avec WooCommerce pour vous connecter de manière sécurisée, récupérer et afficher les transactions commerce de vos clients.

## Configurer l'intégration

Consultez [Configurer l'intégration WooCommerce](/integrations/commerce/woocommerce/commerce-woocommerce-setup) pour apprendre comment configurer et activer l'intégration.

:::caution Extensions et versions

WooCommerce est une plateforme open source conçue pour la configurabilité. Une grande partie de son succès repose sur sa vaste bibliothèque d'extensions proposant des extensions de développeurs tiers. Codat s'efforce d'ajouter le support pour les extensions majeures lorsque c'est possible, mais sachez que dans certains cas, il n'est pas possible de récupérer avec succès les données de marchands avec des extensions non prises en charge.

:::
