:::caution Couverture des types de données

Notre intégration Maxio est disponible pour les clients via proxy uniquement. Veuillez contacter votre gestionnaire de compte pour obtenir l'accès.
:::

:::caution Chargify

Maxio était auparavant connu sous le nom de Chargify. Maxio est une plateforme d'opérations financières qui permet aux équipes B2B SaaS de gérer le GTM, de calculer les revenus et de générer des rapports personnalisés et détaillables. Vous pouvez [en savoir plus sur leur changement de marque ici](https://www.maxio.com/chargify).
:::

<p>
  <a href="https://www.chargify.com/" target="_blank">
    Maxio
  </a>
  est une plateforme de gestion de facturation par abonnement qui permet aux commerçants de
  rationaliser leurs processus de facturation récurrente et de paiement par abonnement.
</p>

Utilisez l'API Commerce de Codat avec Maxio pour vous connecter de manière sécurisée, récupérer et visualiser les ventes récurrentes et les transactions d'abonnement de vos clients.
