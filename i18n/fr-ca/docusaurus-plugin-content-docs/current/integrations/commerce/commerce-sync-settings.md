Mettez à jour vos paramètres de synchronisation pour récupérer automatiquement les données commerce des entreprises liées.

## Mettre à jour les paramètres de synchronisation commerce dans le portail Codat

Dans le portail Codat :

1. Dans la barre de navigation, cliquez sur **Paramètres > Types de données**.
2. Trouvez les types de données suivants et définissez le bouton bascule **Récupérer au premier lien** sur **Activé** :
   - **Commerce - Informations d'entreprise**
   - **Commerce - Commandes**
   - **Commerce - Produits**
   - **Commerce - Litiges**
   - **Commerce - Emplacements**
   - **Commerce - Transactions**
   - **Commerce - Clients**
   - **Commerce - Paiements**
   - **Commerce - Méthodes de paiement**

   Vous pouvez mettre à jour les autres [Paramètres de type de données](/core-concepts/data-type-settings) lorsque vous aurez décidé de la fréquence de synchronisation des données commerce.

3. Sélectionnez **Enregistrer les modifications** pour mettre à jour votre configuration.
