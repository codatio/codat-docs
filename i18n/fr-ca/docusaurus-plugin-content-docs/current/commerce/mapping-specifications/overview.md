Cette section fournit des directives pour la configuration du mappage de données d'un logiciel de commerce vers les logiciels de comptabilité pris en charge (Xero, QuickBooks online).

Vous y trouverez quels types de comptes doivent être sélectionnés pour chaque fonctionnalité et comment les données s'affichent dans le logiciel de comptabilité après une synchronisation réussie via Sync for Commerce.

À des fins de démonstration, elle fait référence à la structure et à l'interface utilisateur de la solution de flux Sync sans code de Codat.

Cependant, Sync for Commerce est un produit flexible qui vous permet de créer une expérience utilisateur similaire ou de concevoir la vôtre pour répondre à vos besoins uniques.

Dans le flux de configuration Sync de Codat, les types de données sont regroupés sous trois fonctionnalités principales :

- Ventes,
- Frais, et
- Paiements.
