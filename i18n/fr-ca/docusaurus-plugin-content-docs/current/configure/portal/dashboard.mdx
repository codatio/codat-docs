## Le portail

La plupart des configurations de votre solution Codat se font via le portail, et vous pouvez effectuer toutes les opérations ci-dessus via les pages de paramètres. Vous pouvez utiliser la barre de navigation pour accéder à ces sections du portail. Certains onglets et pages (par exemple, l'onglet [Développeurs](/configure/portal/developers)) ne sont disponibles que pour des [rôles d'utilisateur](/configure/user-management/user-roles) spécifiques.

### Paramètres

C'est ici que se déroule la majeure partie de votre configuration. Il y a 4 zones de paramètres principales :

- Flux d'authentification - Paramètres relatifs à votre flux d'autorisation, que vous utilisiez Link intégré et hébergé, ou votre propre solution
- Webhooks - Configurez les consommateurs de webhook et vérifiez les types d'événements disponibles
- Intégrations - Activez et configurez les intégrations, et gérez les données que vous récupérez à partir de celles-ci
- Organisation - [Invitez des utilisateurs](/configure/user-management/adding-users) et gérez leurs rôles

### Développeurs

Gérez vos clés API et consultez les ressources spécifiques aux développeurs.
