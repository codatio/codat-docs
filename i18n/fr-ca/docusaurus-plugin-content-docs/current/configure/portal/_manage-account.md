## Gérer le compte

Vous pouvez modifier les détails et les paramètres de votre profil utilisateur dans le [portail Codat](https://app.codat.io/). Cliquez sur le cercle avec vos initiales dans le coin supérieur droit du portail pour accéder à la page [Paramètres du compte](https://app.codat.io/settings/account).

![Capture d'écran du menu déroulant permettant de naviguer vers les paramètres du compte dans le portail Codat](/img/other-guides/0012-acct-mgt-access.png)

Dans la section **Profil**, vous pouvez modifier le nom sur le compte et confirmer si vous êtes impliqué dans l'implémentation technique de votre solution Codat. Cliquez sur **Enregistrer les modifications** lorsque vous avez terminé.

## Changer le mot de passe

Pour changer votre mot de passe, suivez ces étapes dans la section **Mot de passe** de la page [Paramètres du compte](https://app.codat.io/settings/account) :

1. Entrez votre mot de passe actuel dans le champ **Mot de passe actuel**.
2. Entrez votre nouveau mot de passe dans les champs **Nouveau mot de passe** et **Confirmer le nouveau mot de passe**.

   Assurez-vous qu'ils correspondent. Nous vérifierons également la force de votre nouveau mot de passe.

3. Cliquez sur **Mettre à jour le mot de passe** pour enregistrer vos modifications.

![Capture d'écran de la section Mot de passe de la page Paramètres du compte](/img/other-guides/0013-acct-mgmt-password.png)

:::caution Impossible de modifier vos détails ?

L'option de modifier votre nom et votre mot de passe peut être désactivée dans le portail Codat. Pour modifier ces détails, gérez le nom et le mot de passe de la méthode que vous utilisez pour vous connecter (comme Google ou Microsoft).

:::

## Réinitialiser le mot de passe

Pour réinitialiser votre mot de passe, effectuez ces étapes sur l'[écran de connexion du portail Codat](https://app.codat.io/) :

1. Entrez l'adresse courriel que vous utilisez pour vous connecter à Codat et cliquez sur **Continuer**.
2. Cliquez sur **Mot de passe oublié ?** sur l'écran _Entrez votre mot de passe_.
3. Vérifiez que l'adresse courriel indiquée est correcte et cliquez sur **Continuer**.

Vous verrez un écran de confirmation. Ensuite, vérifiez le courriel que vous avez fourni pour les instructions détaillées de réinitialisation du mot de passe.

Si vous utilisez une méthode de connexion différente, telle que Google ou Microsoft, vous devez réinitialiser le mot de passe de cette méthode à la place.
