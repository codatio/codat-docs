Bientôt disponible !

## Dans le portail

Lorsque vous consultez une entreprise dans le portail, vous pouvez afficher l'**Historique de lecture** et l'**Historique d'écriture** pour vérifier le statut des synchronisations de lecture et d'écriture de données précédentes par type de données.

`Entreprises > Entreprise > Historique des données > Historique d'écriture`

Lorsqu'une opération d'écriture de données est initialisée, des éléments apparaîtront ici pour chaque type de données en cours d'écriture.

- Type - nom du type de données, avec le nom de l'intégration en dessous
- ID d'opération d'écriture - l'ID de l'opération d'écriture pour ce type de données
- Date demandée
- Temps pour compléter
- Statut
