## Inviter un nouvel utilisateur

Lorsque vous accédez pour la première fois à Codat, vous serez configuré avec un seul compte Administrateur. Le compte utilise l'adresse courriel du principal point de contact demandé lors de l'intégration. Cet utilisateur Administrateur peut ensuite ajouter de nouveaux utilisateurs avec différents niveaux d'accès.

Pour une explication de ce que signifie chaque niveau d'utilisateur, consultez [Rôles d'utilisateur](/configure/user-management/user-roles).

Pour ajouter de nouveaux utilisateurs depuis le portail Codat :

1. Ouvrez le [portail Codat](https://app.codat.io) et connectez-vous à votre compte Administrateur.
2. Dans la barre de navigation, sélectionnez **Paramètres > Utilisateurs**.
3. Sélectionnez **Inviter un utilisateur** et remplissez le nom et l'adresse courriel de l'utilisateur.
4. Dans la liste **Rôle d'utilisateur**, sélectionnez le [niveau d'autorisation correct](/configure/user-management/user-roles) pour l'utilisateur.
5. Pour terminer la configuration du compte, sélectionnez **Envoyer l'invitation**.

Un courriel est envoyé à l'utilisateur du nouveau compte pour qu'il vérifie les détails de son compte et configure un mot de passe.

## Modifier les détails d'un compte existant

À tout moment, les utilisateurs ayant un accès Administrateur peuvent mettre à jour les détails de compte de tous les utilisateurs, y compris leur nom, leur adresse courriel et leur niveau d'accès. Les administrateurs peuvent également activer ou désactiver des comptes.

1. Ouvrez le [portail Codat](https://app.codat.io) et connectez-vous à votre compte Administrateur.
2. Dans la barre de navigation, sélectionnez **Paramètres > Utilisateurs**.
3. Trouvez les détails de l'utilisateur qui vous intéressent et sélectionnez **Modifier** à côté d'eux.
4. Modifiez tous les détails nécessaires, puis sélectionnez **Enregistrer les modifications**.
