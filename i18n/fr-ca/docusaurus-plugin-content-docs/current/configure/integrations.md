## Configuration des intégrations

Vous pouvez activer les intégrations dont vous avez besoin [via le portail](https://app.codat.io/settings). Chaque page de configuration d'intégration montre comment configurer cette intégration, y compris des liens vers la documentation externe et le portail développeur de l'intégration.

Consultez notre [documentation](/integrations/all-integrations) détaillée spécifique aux intégrations pour plus d'informations sur les intégrations dont vous avez besoin.

### Données comptables

Pour gérer quelles intégrations de logiciels de comptabilité sont activées, utilisez le portail Codat :

- Cliquez sur [**Paramètres > Intégrations > Comptabilité**](https://app.codat.io/settings/integrations/accounting)
- Sélectionnez l'intégration que vous souhaitez gérer
- Configurez et activez l'intégration

Des guides sur la configuration de chaque intégration sont disponibles dans l'[aperçu des intégrations comptables](/integrations/accounting/overview).

### Données bancaires

Pour gérer quelles intégrations de comptes bancaires sont activées, utilisez le portail Codat :

- Cliquez sur [**Paramètres > Intégrations > Bancaire**](https://app.codat.io/settings/integrations/banking)
- Sélectionnez l'intégration que vous souhaitez gérer
- Configurez et activez l'intégration

Nous recommandons d'utiliser une seule intégration bancaire à la fois.

Des guides sur la configuration de chaque intégration sont disponibles dans l'[aperçu des intégrations bancaires](/integrations/banking/overview).

### Données commerciales

Pour gérer quels logiciels de comptabilité sont activés, utilisez le portail Codat :

- Cliquez sur [**Paramètres > Intégrations > Commerce**](https://app.codat.io/settings/integrations/commerce)
- Sélectionnez l'intégration que vous souhaitez gérer
- Configurez et activez l'intégration

D'autres guides détaillés sur la configuration de celles-ci sont disponibles dans l'[aperçu des intégrations commerciales](/integrations/commerce/overview).
