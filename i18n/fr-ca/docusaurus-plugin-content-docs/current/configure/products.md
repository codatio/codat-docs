## Activation des solutions

Lors de l'inscription à un compte Codat, vous devrez sélectionner et activer une solution qui vous intéresse (vous pouvez toujours activer d'autres solutions plus tard).

L'activation d'une solution configurera le portail et votre nouveau client pour l'utilisation de cette solution, y compris les paramètres de type de données par défaut.

Vous pouvez gérer vos solutions activées dans [le portail](https://app.codat.io/settings/products) (utilisateurs avec rôle d'administrateur uniquement). Actuellement, vous ne pouvez pas désactiver les solutions - veuillez contacter le support si vous avez besoin d'aide à ce sujet.
