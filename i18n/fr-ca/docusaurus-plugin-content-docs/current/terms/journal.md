title: Journal
hoverText: Les journaux sont utilisés pour enregistrer toutes les transactions financières d'une entreprise.
---

Dans les logiciels comptables, les journaux sont utilisés pour enregistrer toutes les transactions financières d'une entreprise. Chaque transaction dans un journal est représentée par une écriture de journal distincte. Ces écritures sont utilisées pour créer le grand livre général, qui est ensuite utilisé pour créer les états financiers d'une entreprise.

Lorsqu'une entreprise enregistre toutes ses transactions dans un seul journal, celui-ci peut devenir volumineux et difficile à maintenir et à suivre. C'est pourquoi les grandes entreprises utilisent souvent plusieurs journaux (également appelés sous-journaux) pour catégoriser et gérer les écritures de journal.

Ces journaux peuvent être divisés en deux catégories :

- Journaux spéciaux : journaux utilisés pour enregistrer des types spécifiques de transactions ; par exemple, un journal des achats, un journal des ventes ou un journal de gestion de trésorerie.
- Journaux généraux : journaux utilisés pour enregistrer les transactions qui ne relèvent pas de la portée des journaux spéciaux.
