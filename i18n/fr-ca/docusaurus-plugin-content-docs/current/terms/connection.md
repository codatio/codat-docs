---
title: Connexion
hoverText: Les connexions représentent les liens d'une entreprise à une source de données qui vous permettent de lire et d'écrire des données vers cette source.
---

Une connexion de données représente la connexion d'une entreprise à une source de données et vous permet de synchroniser des données (lire ou écrire) avec cette source.

Une entreprise peut avoir plusieurs connexions de données selon le type de source de données à laquelle elle se connecte. Par exemple, une seule entreprise peut se lier à :

- Données comptables - 1 connexion active.
- Données bancaires - Plusieurs connexions actives.
- Données de commerce - Plusieurs connexions actives.

Toute combinaison de connexions de données comptables, bancaires et de commerce est autorisée.
