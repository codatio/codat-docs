Les comptes sont les catégories qu'une entreprise utilise pour enregistrer les transactions comptables. À partir des endpoints Comptes, vous pouvez récupérer une liste de tous les comptes pour une entreprise spécifiée.

Les comptes sont également appelés plan comptable, comptes nominaux et grand livre général.
