Dans Codat, une intégration représente une source de données tierce à laquelle vous pouvez connecter une [entreprise](/core-concepts/companies) (en utilisant une [connexion de données](/core-concepts/connections)) afin de pouvoir accéder aux données commerciales de cette entreprise.

Vous pouvez connecter une entreprise à une seule source comptable et une seule source de flux bancaires, et à plusieurs plateformes de commerce et bancaires.
