Dans Codat, chaque type de données que vous souhaitez créer, mettre à jour ou récupérer pour une entreprise donnée à partir d'une source de données est représenté par un type de données. Il a un schéma défini avec des propriétés connues et des règles de validation.

Les types de données que vous utilisez dépendront de la solution Codat que vous utilisez. Pour référence, notre solution [Lending](/lending/data-types) offre la plus large gamme de types de données.
