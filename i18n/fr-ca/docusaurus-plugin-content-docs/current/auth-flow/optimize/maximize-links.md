Lier des données
Qu'est-ce que l'autorisation ?

Le processus d'autorisation est le mécanisme par lequel vos clients PME partagent leurs données financières avec vous en liant leurs données. Une fois qu'une entreprise a lié ses données comptables, bancaires ou commerciales, vous pouvez utiliser ces données pour améliorer votre entreprise.

Lier des données avec vous offre des avantages à la fois pour la PME qui lie ses données et pour vous en tant que prêteur. Ce guide présente des suggestions et des conseils pour maximiser le nombre d'entreprises qui partagent leurs données avec vous, et maximise ainsi l'avantage pour vous et vos clients.

Importance de maximiser les liens

Permet un accès facile et continu aux données financières, ce qui :

économise votre temps et votre argent
réduit votre risque grâce à l'accès à des données en temps réel
vous permet de prioriser les entreprises qui nécessitent une attention maintenant
améliore l'expérience pour vos clients
augmente vos ventes/débit et accorde un avantage concurrentiel global
Échange de valeur

L'échange de valeur fondamental du processus de liaison est qu'une entreprise partage des données financières avec vous pour recevoir les meilleurs services disponibles pour ses besoins spécifiques. L'échange de valeur est formé d'avantages pour vous en tant que prêteur et pour votre client.

Avantage pour vous en tant que prêteur :
L'accès aux données financières vous permet de comprendre la position de vos clients
Les données sont disponibles dans l'interface utilisateur Codat, via API ou téléchargement Excel au lieu d'e-mails ou de PDF manuels
Les données sont dans un format standard provenant de nombreuses sources, permettant une facilité de compréhension
Les avantages pour votre client vous profitent également
Avantage pour votre client :
Les clients recevront de meilleurs services parce que vous comprenez leur position plus en détail
Le parcours de liaison simple élimine le besoin de créer manuellement des rapports et de les télécharger ou de les envoyer par e-mail
Les données requises sont rassemblées en une seule fois, réduisant le besoin de demander plus de données
Maximiser les liens

Maximiser le nombre d'entreprises qui choisissent de lier leurs données maximise les avantages que vous recevez de Codat. Il y a un certain nombre de choses que vous pouvez faire pour maximiser le nombre d'entreprises qui se lient.

Considérations sur le parcours utilisateur

L'intégration du flux de liaison dans votre propre parcours d'application a permis plus de liens que les approches manuelles parmi les clients de Codat à ce jour. En gardant cela à l'esprit, les directives pour le portail Codat et pour l'implémentation API sont ci-dessous.

API

L'implémentation API du flux de liaison implique que vous construisiez votre propre parcours de liaison alimenté par l'API Codat. Conseils pour maximiser les liens :

Les flux de consentement intégrés dans votre propre parcours d'application et en accord avec votre marque sont plus efficaces
Le parcours doit être aussi court que possible sans compromettre d'autres facteurs
La solution sans code Codat permet à une entreprise de se lier en 6 clics
Fixez l'attente du temps requis à l'avance – une entreprise est plus susceptible de se lier si elle sait que le processus ne prend qu'une minute
Le parcours doit être intuitif et inciter l'utilisateur à l'étape suivante à chaque fois
Envisagez de faire apparaître la capacité de supprimer les connexions dans votre flux
Envisagez d'ajouter une section FAQ sur le partage de données dans le cadre de votre flux
Placez l'option de lier des données financières près du point où l'entreprise comprend qu'elle a besoin de votre produit
Par exemple, un compte bancaire montrant un découvert montre une offre pour une solution de financement à plus long terme pour l'entreprise en partageant des données financières
Remarque : certaines entreprises ne veulent pas voir d'offres non sollicitées
Simplifiez les flux grâce à des éléments visuels tels que des icônes
Portail

L'implémentation du portail du flux de liaison implique que vous envoyiez manuellement l'URL de liaison à l'entreprise. Conseils pour maximiser les liens :

Identifiez les groupes cibles et envoyez le lien d'introduction général Codat à tous les clients ou prospects de ce groupe
Précisez clairement que l'entreprise peut révoquer le consentement à tout moment dans son logiciel de comptabilité
Envisagez d'intégrer le flux d'autorisation dans votre parcours d'application
Considérations sur la messagerie

Quelle que soit la méthode d'implémentation, les clients voudront comprendre ce qu'ils font lorsqu'ils partagent des données via le processus de liaison et les avantages de le faire. La messagerie autour du flux de liaison est un facteur important pour maximiser les liens. Les clients sont plus susceptibles de lier leurs données si vous :

Expliquez quel est le but du partage de données et comment les données seront utilisées
Indiquez si le partage de données via le flux de liaison est obligatoire ou facultatif
Expliquez la nécessité des données pour le produit ou service spécifique considéré
Nous conseillons de faire de cela une étape obligatoire, bien que les tests A/B soient également recommandés pour s'assurer que cela n'affecte pas votre pipeline
Expliquez qui aura accès aux données et comment vous les garderez en sécurité
Expliquez comment ils peuvent révoquer le consentement ou changer d'avis
Échange de valeur et incitations

Une entreprise est plus susceptible de partager ses données si elle a une incitation tangible. Les incitations financières sont de loin les plus efficaces. Une incitation peut également prendre la forme du client comprenant les avantages qu'il reçoit. Suggestions pour aborder cette perspective :

Identifiez le moteur pour l'application particulière
Pour un besoin d'argent en temps désespéré - l'échange de valeur peut être un traitement plus rapide et un délai plus court pour l'argent
Pour un besoin d'argent pour une nouvelle opportunité - des délais de traitement plus longs utilisant des méthodes manuelles peuvent entraîner une opportunité manquée et la concurrence pouvant potentiellement entrer
Pour les facilités renouvelables (telles que le financement de factures) - la suppression des processus manuels peut s'assurer qu'ils obtiennent les fonds à temps chaque période, éliminant le risque de soumissions oubliées pour une période donnée
Covenants - supprimez le risque qu'ils manquent la période d'audit/de surveillance des covenants. Manquer la période de rapport peut entraîner le défaut de l'entreprise sur son accord et pourrait entraîner des conséquences négatives telles qu'un accès limité à un financement supplémentaire, un impact sur le score de crédit
Économie de temps quantifiable : un processus manuel peut vous prendre quelques heures chaque mois, et le processus de liaison vous fait gagner ce temps
Offrez des services plus personnalisés aux clients qui utilisent le processus de liaison
Le partage manuel de données financières ne sera pas la norme dans les années à venir – informez le client qu'il a une longueur d'avance
Au premier point d'engagement, les données sont extraites manuellement dans le format requis, ce qui signifie qu'ils n'ont pas à télécharger à partir de différents systèmes et à formater, ils peuvent passer du temps concentré sur leur entreprise et à la rendre prospère.
Meilleurs taux - si vous économisez de l'argent sur l'intégration initiale/le service continu/la réduction des créances irrécouvrables, pouvez-vous transmettre une partie de cet avantage au client ?
Le processus de liaison permet aux petites entreprises d'être servies à des taux compétitifs ; en utilisant des processus manuels, ce segment n'est pas rentable à servir
Envisagez d'utiliser la fonctionnalité d'écriture de Codat pour saisir les transactions de prêt dans le logiciel de comptabilité en leur nom, réduisant encore le travail manuel pour votre client
