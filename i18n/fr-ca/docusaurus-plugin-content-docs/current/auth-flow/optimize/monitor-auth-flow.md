Nous vous équipons d'outils pour surveiller votre flux d'autorisation. Dans le [portail Codat](https://app.codat.io/monitor/auth-flow-overview), vous pouvez voir où vos clients rencontrent des difficultés et optimiser le flux si nécessaire. Cette surveillance est effectuée dans toutes les catégories d'intégration activées.

![](/img/link/0002-auth-flow-monitor.png)

[Naviguez](https://app.codat.io/monitor/auth-flow-overview) vers **Surveiller > Flux d'autorisation > Aperçu** pour découvrir le **pourcentage** global de vos clients qui ont autorisé les connexions d'intégration et le pourcentage global d'entreprises qui ont téléchargé un fichier dans le cadre de leur processus d'autorisation.

### Entonnoir d'autorisation

Analysez l'**entonnoir d'autorisation** pour voir combien de clients ont terminé leur parcours d'autorisation par rapport à la quantité de ceux qui ont commencé le flux Link et de ceux qui ont commencé le processus d'autorisation.

:::info Étapes optionnelles

Si une catégorie d'intégration est marquée comme optionnelle dans **Paramètres > Flux d'autorisation > Link**, sauter cette étape augmente la mesure de réussite de l'autorisation de l'entonnoir.
:::

:::info Vous utilisez votre propre flux d'autorisation ?

Si vous avez créé votre propre solution, nous ne sommes pas en mesure de collecter des données pour l'étape **A commencé le flux**. Au lieu de cela, nous estimerons cette étape dans l'entonnoir en fonction du nombre d'entreprises qui passent à l'étape suivante. Tous les autres points de données devraient fonctionner comme prévu.
:::

### Taux de complétion au fil du temps

Visualisez vos statistiques de complétion du flux d'autorisation à l'aide du **graphique**. Il représente le rapport entre le nombre de parcours d'autorisation terminés et le nombre de flux d'autorisation commencés au cours d'une semaine donnée, exprimé en pourcentage.

:::caution Vous ne voyez aucune donnée ?

La surveillance ne commence qu'une fois que votre premier client s'est autorisé avec succès. Par conséquent, les premiers jours d'utilisation peuvent ne pas apparaître sur le graphique de conversion si aucun parcours n'a été terminé.
:::
