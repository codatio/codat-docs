---
title: "L'entonnoir"
description: "Augmentez votre taux de conversion en permettant aux clients de se connecter rapidement"
---

Les étapes menant au point de connexion doivent être réduites au minimum. Le parcours menant à la connexion ne doit pas être lent et ardu. Il doit être simple, communiquer clairement la valeur de la connexion des données et permettre à votre client de se connecter aussi rapidement que possible :

- Encouragez à se connecter dans l'e-mail de bienvenue
- Demandez à vos utilisateurs de se connecter en 3 actions maximum à partir de l'inscription
- Assurez-vous que l'appel à l'action pour se connecter est visible pour les utilisateurs
- Utilisez la preuve sociale pour accroître la confiance dans votre solution
- Bonus : automatisez le remplissage de toutes les pages de configuration avec les données de l'entreprise connectée

:::success Pourquoi c'est important

Selon nos données, vous pouvez **augmenter le taux de conversion de plus de 30%** avec un entonnoir bien construit qui permet à vos clients de se connecter rapidement.
:::

## Encouragez à se connecter dans l'e-mail de bienvenue

Votre e-mail de bienvenue ou d'inscription doit inclure plusieurs appels à l'action (CTA) qui encouragent votre client à connecter ses comptes. Par exemple, envisagez d'afficher un simple compteur en 3 étapes avec la connexion des comptes comme deuxième étape « déblocage de valeur ».

## Demandez à vos utilisateurs de se connecter en 3 actions maximum à partir de l'inscription

Demander à vos utilisateurs de se connecter en 3 étapes/clics maximum à partir du point d'atterrissage sur votre site web maximise la probabilité de connexion lors du premier parcours et dans l'ensemble. Assurez-vous que les étapes menant à la connexion sont aussi simples que possible.

Considérez cet exemple :

1. Un utilisateur s'inscrit.
2. L'e-mail de bienvenue comprend de grands CTA pour connecter leurs comptes, avec des étapes simples « 1, 2, 3 » décrites pour la configuration, l'une des étapes étant de connecter leurs comptes.
3. L'utilisateur est immédiatement invité à connecter leur logiciel clé, qui peut également être utilisé pour pré-remplir tous les formulaires afin de réduire la saisie manuelle pour votre client.

## Assurez-vous que l'appel à l'action pour se connecter est visible pour les utilisateurs

Nous recommandons de placer l'appel à l'action pour se connecter dans la section centrale de l'interface utilisateur de votre application ou d'en faire une option permanente du menu principal. Si vous l'incluez dans un sous-menu, vos utilisateurs auront du mal à le trouver, ce qui entraînera une connectivité plus faible.

## Utilisez la preuve sociale pour accroître la confiance dans votre solution

Envisagez d'afficher un avis positif d'une personne de confiance, par exemple, une personne influente qui défend votre produit dans l'une des entreprises de vos clients.

## ⭐ Automatisez le remplissage des pages de configuration avec les données de l'entreprise connectée

Plus un client connecte ses données tôt dans l'entonnoir, plus vous pouvez automatiser son flux d'intégration en pré-remplissant [des informations utiles sur l'entreprise](/accounting-api#/schemas/CompanyDataset) telles que l'enregistrement de l'entreprise, le numéro de taxe, les adresses et les informations de contact clés ou [le chiffre d'affaires annuel de l'entreprise](/accounting-api#/schemas/balancesheet).
