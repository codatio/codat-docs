## Aperçu

Link est une solution en marque blanche qui peut être adaptée à la marque de votre entreprise. Accédez à **[Paramètres > Flux d'autorisation > Marque](https://app.codat.io/settings/branding)** dans le [portail Codat](https://app.codat.io) pour personnaliser Link pour une expérience entièrement personnalisée.

La page des paramètres de marque s'enregistre automatiquement lorsque des modifications sont apportées. Vous pouvez ajuster les éléments d'interface utilisateur suivants :

### Nom de l'entreprise

Le nom de votre entreprise, tel que vous le voyez dans le portail, est également affiché dans Link lorsque vos clients connectent leur logiciel de comptabilité à Codat.

### Logo de l'entreprise

Ce logo apparaît dans la barre de navigation de Link et sur les écrans de connecteur sur site lorsque les utilisateurs lient des applications sur site (par exemple, Sage 50 et QuickBooks Desktop).

Exigences pour le logo :

- Utilisez les formats de fichier PNG, JPEG, SVG ou GIF.
- Les fichiers ne doivent pas dépasser 10 Mo.
- Dimensions recommandées : 600px x 100px.

### Icône de l'entreprise

Cette icône (téléchargée sous forme de favicon) apparaît comme l'icône de l'onglet du navigateur Link et sur l'icône de la barre d'état système des connecteurs sur site lorsqu'une application sur site s'exécute en arrière-plan sur l'ordinateur d'un client.

Exigences pour l'icône :

- Utilisez le format de fichier ICO.
- Les fichiers ne doivent pas dépasser 500 Ko.
- Dimensions recommandées : 32px x 32px.

### Couleur primaire

Vous pouvez définir la couleur primaire de votre marque pour définir la couleur des boutons et d'autres éléments dans le flux Link. Nous recommandons d'utiliser une couleur primaire foncée, car les couleurs claires peuvent causer des problèmes d'accessibilité.
